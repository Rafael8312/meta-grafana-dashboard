# Meta Grafana Dashboard

Dashboard de performance Meta Ads com Grafana - Insights em tempo real de campanhas, conjuntos e anúncios.

## Funcionalidades

- ✅ Coleta automática de dados da Meta Marketing API a cada 30 minutos
- ✅ Métricas de "Conversas por mensagem iniciadas" (WhatsApp)
- ✅ Cálculo de CPL (Custo por Lead)
- ✅ Visualização de "últimos 30 minutos" por diferença de snapshots
- ✅ Integração com Grafana Cloud via Infinity datasource
- ✅ Deploy no Render.com (free tier)
